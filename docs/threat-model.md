# Threat model

This document lists possible attacks and privacy risks.

Initial risks to consider:

- Server learns the friendship graph.
- Server learns how many friends a user has.
- Sender identity leaks through message metadata.
- BLE advertisements reveal stable identifiers.
- QR pairing data can be forged or replayed.
- Location is shared too precisely or too often.
- Spam or denial-of-service attacks abuse the BeerMe signal.
